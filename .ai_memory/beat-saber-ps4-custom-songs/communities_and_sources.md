# Beat Saber Custom Song Communities, Sources & Tools

## Primary Song Repositories

| Name | URL | Type | Description |
|------|-----|------|-------------|
| **BeatSaver** | https://beatsaver.com | Repository | The primary and most comprehensive repository for all Beat Saber custom maps. Every published map is hosted here. Features search, filtering, OneClick install, API access. |
| **BeastSaber** | https://bsaber.com | Community Hub | Community-curated songs, featured packs, Map of the Week, player reviews and ratings. Pulls data from BeatSaver. Great for discovering quality maps. |
| **ScoreSaber** | https://scoresaber.com | Leaderboards | Competitive leaderboard system for custom songs. Indicates ranked and popular songs with difficulty ratings. |
| **BSSB (Beat Saber Server Browser)** | https://bssb.app | Statistics | Top 100 most-played custom levels based on multiplayer play count. Playlist downloads available. |
| **bs-map-converter** | https://github.com/airalmer/bs-map-converter | Tool | Web-based tool that converts v4 maps to v3 format (useful for older song compatibility). |

## Desktop Management Tools

| Name | URL | Platform | Description |
|------|-----|----------|-------------|
| **ModAssistant** | (desktop app) | PC / Quest | One-click installer for mods and songs. Integrates with BeatSaver and other sources. |
| **BeatDrop** | https://github.com/StarGazer1258/BeatDrop | PC | Desktop app for downloading and managing mods, songs, and playlists for PC/Quest Beat Saber. |
| **BSManager** | https://github.com/Zagrios/bs-manager | PC | Centralized Beat Saber manager for mods, maps, and game versions. Recommended mod installer. |
| **QuestSongManager** | https://github.com/exelix11/QuestSongManager | Quest | Song and playlist manager for Meta Quest. Syncs BeatSaver playlists, OneClick downloads. |

| Name | URL | Description |
|------|-----|-------------|
| **PSXHAX** | https://www.psxhax.com | Primary PS4/PS5 scene forum. Hosts Beat Saber FPKG game dumps and DLC packs by Opoisso893. |

## Development / Reverse Engineering

| Name | URL | Description |
|------|-----|-------------|
| **UABE** | https://github.com/SeriousCache/UABE | Unity Asset Bundle Extractor for editing `.assets` and AssetBundle files. Essential for extracting/replacing audio and beatmap data. |
| **UABEA** | https://github.com/nesrak1/UABEA | Modern cross-platform fork of UABE, supports newer Unity versions. |
| **bsmg/Beat Saber Wiki** | https://bsmg.wiki | Official modding group wiki. Map format documentation, installation guides, mod info. |
| **BSMG Map Format Docs** | https://bsmg.wiki/mapping/map-format.html | Detailed schema documentation for Beat Saber map files (v1-v4 formats). |
| **beatsabertools/saberschema** | https://github.com/beatsabertools/saberschema | JSON schemas for Beat Saber map files. |
| **Trishume Hex Fiend Template** | https://gist.github.com/trishume/138ef8f6c66fabd2d76b9fdf8d5c4c67 | Binary format template for reverse-engineering Beat Saber Unity assets. |
| **beat_saber_map (Rust)** | https://docs.rs/beat_saber_map/latest/beat_saber_map/ | Rust library for parsing Beat Saber map formats (all versions). |
| **MCJack123 Format Conversion** | https://gist.github.com/MCJack123/892b936ef0d18da43ab2764ba97402be | Info.dat v1 (JSON) to v2 (binary) conversion reference. |
