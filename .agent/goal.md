# Project: Beat Saber PS4 Custom / Community Song Support

The PC / Oculus VR version of Beat Saber has a rich community online that creates custom song levels for the game and shares them.

I have the PS4 VR version of Beat Saber and a jailbroken PS4. I'd like to explore the possibility of taking custom community Beat Saber PC songs and converting them into a format that is installable on the PS4 (in a PKG file) so that I can play these custom songs in the game.

I do something similar with Rock Band 4 on my PS4, using the Rock Band Deluxe mod and the Onyx toolkit and LibForge / ForgeTool. This works quite well to let me install custom / community songs.

## Suggested end result / operational details

There are three potential methods by which we can "insert" the custom songs into the Beat Saber PS4 GUI so they are selectable and playable. These are listed in order of perceived effort from most to least (optimal / simplest solution is the last one). We will want to try and achieve #3 first and evaluate if that is good enough before moving up the list here. #3 might be enough to call complete to release this to the community.

1. Add a new "Album" to the Song Selection Screen with each custom PKG full of songs we install
2. Add a single "Album" to the Song Selection Screen that will contain all custom songs we ever install
3. Insert custom songs all into an existing "Album" already represented in the PS4 Beat Saber GUI, such as "Extras"
