# Project: Beat Saber PS4 Custom / Community Song Support

The PC / Oculus VR version of Beat Saber has a rich community online that creates custom song levels for the game and shares them.

I have the PS4 VR version of Beat Saber and a jailbroken PS4. I'd like to explore the possibility of taking custom community Beat Saber PC songs and converting them into a format that is installable on the PS4 (in a PKG file) so that I can play these custom songs in the game.

I do something similar with Rock Band 4 on my PS4, using the Rock Band Deluxe mod and the Onyx toolkit and LibForge / ForgeTool. This works quite well to let me install custom / community songs.

Avenues to explore:

- Locate communities / forums online where custom Beat Saber songs are discussed and shared, catalog the sources / communities / websites in a markdown file with links. Also create another markdown file with all the custom Beat Saber songs you can find links to, in a table (song name / artist / album / beat saber author / download link)

- Search for a decompilation project for Beat Saber PS4 that might help us

- Search for any active projects that might be trying to do the same thing we are here, is there an existing way to add custom songs to Beat Saber PS4?

- Theorize / hypothesize how else we might proceed and perform as many investigations as you can to gather information.
