## Files to Copy to USB

Copy these **2 files** to USB drive root:

1. `UP8802-CUSA12878_00-BSCUSTOMSONGS01-unlock.pkg` (in output folder)
2. `test_10songs.pkg` (or `custom_songs.pkg`)

---

## Installation Order

### Step 1: Install Unlocker FIRST
1. Plug USB into PS4
2. Open **GoldHEN**
3. **Install Package** > **Install Package Files**
4. Select `UP8802-CUSA12878_00-BSCUSTOMSONGS01-unlock.pkg`
5. Wait for "Installation Complete"

### Step 2: Install Songs SECOND
1. Same menu: **Install Package Files**
2. Select `test_10songs.pkg`
3. Wait for "Installation Complete"

---

## Finding Custom Songs In Game

- Launch **Beat Saber**
- Go to **Play** > **Campaign** or **Solo**
- Scroll down - custom songs appear at bottom of song list
